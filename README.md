## 🎯 Overview

**InterviewIQ AI** is a production-ready AI-powered interview coaching platform designed to simulate real technical interviews. It uses **Groq API** with **Llama 3.3 70B** to generate role-specific questions, evaluate answers across 5 dimensions, provide instant feedback, and give career recommendations.

> **MCA GenAI Final Year Project** — Built with modern web technologies and real AI integration.

---

## ✨ Features

| Feature | Description |
|---------|-------------|
| 🤖 **AI Interview Simulation** | Role-specific questions generated by Llama 3.3 70B |
| ⚡ **Instant Evaluation** | 5-dimension scoring: Technical, Clarity, Depth, Communication, Problem Solving |
| 📊 **Performance Analytics** | Radar charts, line charts, bar charts, pie charts |

| 🌓 **Dark/Light Mode** | Premium dark glassmorphism design |
| 📱 **Responsive Design** | Mobile-friendly across all devices |
| 🏆 **Interview History** | Track performance across all interviews |
| 🔐 **JWT Authentication** | Secure user registration and login |

## 🚀 Installation Guide

### Prerequisites
- Python 3.11+
- Node.js 18+
- PostgreSQL 14+ (or Docker)
- Groq API Key

### Option A: Run with Docker (Recommended for PostgreSQL)

```bash
# 1. Clone the repository
git clone https://github.com/yourusername/interviewiq-ai.git
cd interviewiq-ai

# 2. Copy environment file
cp .env.example .env
# Edit .env and add your GROQ_API_KEY

# 3. Start PostgreSQL with Docker
docker-compose up -d db

# 4. Setup Backend
cd backend
pip install -r requirements.txt
uvicorn app.main:app --reload
```

### Option B: Manual Setup

#### Backend Setup
```bash
# Navigate to backend
cd backend

# Create virtual environment
python -m venv venv

# Activate (Windows)
venv\Scripts\activate
# Activate (Linux/Mac)
source venv/bin/activate

# Install dependencies
pip install -r requirements.txt

# Create .env file (copy from .env.example)
copy .env.example .env

# Run the server
uvicorn app.main:app --reload --port 8000
```

The API will be available at: http://localhost:8000
Swagger Docs: http://localhost:8000/docs

#### Frontend Setup
```bash
# Navigate to frontend
cd frontend

# Install dependencies
npm install

# Start development server
npm run dev
```

The app will be available at: http://localhost:5173

## 🔐 Environment Variables

Create a `.env` file in the `backend/` directory:

```env
# Groq AI Configuration
GROQ_API_KEY=your_groq_api_key_here
MODEL_NAME=llama-3.3-70b-versatile

# Database
DATABASE_URL=postgresql://postgres:password@localhost:5432/interviewiq

# JWT Authentication
JWT_SECRET=your-super-secret-key-here-make-it-long
JWT_ALGORITHM=HS256
ACCESS_TOKEN_EXPIRE_MINUTES=1440
```

## 🐘 PostgreSQL Setup

### Using Docker (Easiest)
```bash
# Start only the database
docker-compose up -d db

# Verify it's running
docker-compose ps

# Connect to database
docker exec -it interviewiq_db psql -U postgres -d interviewiq
```

### Manual PostgreSQL Setup
```sql
-- Connect to PostgreSQL
psql -U postgres

-- Create database
CREATE DATABASE interviewiq;

-- Create user (optional)
CREATE USER interviewiq_user WITH PASSWORD 'your_password';
GRANT ALL PRIVILEGES ON DATABASE interviewiq TO interviewiq_user;
```

The tables are **auto-created** when you start the FastAPI server (SQLAlchemy `create_all`).

## 🤖 Groq API Setup

1. Visit [console.groq.com](https://console.groq.com)
2. Sign up / Log in
3. Navigate to **API Keys** → **Create API Key**
4. Copy the key and add to your `.env` file as `GROQ_API_KEY`

**Model used:** `llama-3.3-70b-versatile`
- Context: 128K tokens
- Speed: Ultra-fast inference (~300 tokens/second)
- Cost: Free tier available

## 📡 API Documentation

Visit http://localhost:8000/docs for interactive Swagger UI.

### Authentication Endpoints

| Method | Endpoint | Description |
|--------|----------|-------------|
| POST | `/auth/register` | Register new user |
| POST | `/auth/login` | Login user, returns JWT |
| GET | `/auth/me` | Get current user profile |

### Interview Endpoints

| Method | Endpoint | Description |
|--------|----------|-------------|
| POST | `/interviews/` | Create interview + generate questions |
| GET | `/interviews/` | List all user interviews |
| GET | `/interviews/{id}` | Get interview details |
| POST | `/interviews/{id}/submit-answer` | Submit answer + get evaluation |
| POST | `/interviews/{id}/complete` | Complete interview, get final score |
| GET | `/interviews/{id}/report` | Get full interview report |
| GET | `/interviews/{id}/pdf` | Download PDF report |
| GET | `/interviews/{id}/career` | Get career recommendations |
| GET | `/interviews/stats/dashboard` | Get dashboard statistics |

### Resume Endpoint

| Method | Endpoint | Description |
|--------|----------|-------------|
| POST | `/resume/analyze` | Upload PDF resume + generate questions |
